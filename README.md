# Moon Weeks

This is Rimworld game mod which generate weekly themed events.

Each week can:
- add pawn hediffs
- add pawn thought
- add traits
- cause mental breaks
- apply inspirations
- change weather
- change map plant and animal density
- start other events and weeks.
